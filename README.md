# React Native PoC

# Usage

```
react-native run-android
```

ou

```
react-native run-ios
```

Le bundler peut être lancé indépendamment

```
react-native start
```

# Uses

* babel:
  * .babelrc
* eslint:
  * .eslintrc
* flow:
  * .flowconfig
* prettier:
  * package.json
