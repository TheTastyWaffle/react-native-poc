# React Native Betting Client

Client du paris Winamax en ligne.

* Cibles: Android, iOS
* Cibles à venir: Web

# Usage

```
react-native run-android
```

ou

```
react-native run-ios
```

Le bundler peut être lancé indépendamment

```
react-native start
```
